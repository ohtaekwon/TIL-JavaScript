###  :pencil: ​_문제 005 : for문 계산_

다음 코드의 출력 값으로 알맞은 것은?

```javascript
var a = 10;
var b = 2;

for(var i=1; i<5; i+=2){
    a += i;
}
console.log(log(a+b)); // 16
```

> **1 ) 10**
>
> **2 ) 12**
>
> **3 ) 14**
>
> **4 ) 16**

<br>

:ballot_box_with_check: **풀이 해석**

```javascript
var a = 10;
var b = 2;

for(var i=1; i<5; i+=2){
    // i+=2는 i=i+2 의 축약 코드
    // a = a + i의 축약코드
    a += i;
}
console.log(log(a+b)); // 16
```

- 첫 번째 일때, i = 1
- 두 번째 반복되었을 때, i = 3
- 세 번째 반복되었을 때, i =5가 된다. 하지만, i < 5의 조건식이 있으므로 for문은 종료가 된다.
- ***최종적으로 for문은 2번만 반복이 된다.***

<br>

| i =     | a =  | 반복횟수 |
| ------- | ---- | -------- |
| 1 (1<5) | 11   | 첫 번째  |
| 3 (3<5) | 14   | 두 번째  |
| 5 (5<5) | x    | 세 번째  |

- a의 최종값은 14
- b의 최종값 2
