# PART 2

###  :pencil: ***입문 022 :  비교연산자***

<br>

비교 연산자는 두 개의 값을 비교하여 *true*와 *false* 결과값을 반환한다. 비교 연산자의 종류에는 값이 동등한지 비교하는 **일치연산자**와 값의 관계를 비교하는 **관계 연산자**가 이싿. 

**일치 연산자**는 **값의 일치 여부**를 확인하며, 종류에는 `동등 연산자`, `부등 연산자`, `일치 연산자`, `불일치 연산자`가 있다. 

```javascript
console.log(5==5);
console.log("5"==5);
console.log(5=="5");
console.log(5!=1);
```
